My NixOS Configuration

This project was started with the help of two very good friends of mine. For context, NixOS (no relation to me) is a free and open-source Linux distribution known for its reproducibility and portability. This is achieved by using immutable design, and an atomic update model. The distribution uses the Nix language, which is used in packages recipes and configurations. The language itself is functional, meaning that each nix file is a function, that can be imported for use in other such files [1]. For more information, feel free to check out my repository here.
